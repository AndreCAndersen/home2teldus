    Google Home IFTTT Telldus Live!

    This is a free API service to enable voice commands from Google Home to Telldus Live! via IFTTT. It was created primarly to support my own DYI home automation setup, but as it is already running it might as well respond to external requests as well.

    Instruction and source code can be found on Github.
